Sobre o uso de imagens

É muito importante saber que nem todas as imagens na web podem ser utilizadas livremente na construção de sites. No Google Imagens, é importante buscar na aba "ferramentas" imagens livres para uso comercial, pois elas podem ser alteradas. Alguns sites que possuem imagens livres para download e uso são o Unsplash, Pexels e Pixabay.

Os formatos de imagens para web mais usados são o JPEG e PNG.

JPEG: melhor formato de compactação, para que as imagens fiquem mais leves, favorecendo o carregamento do site.

PNG: permite transparência no fundo das imagens e animação nas imagens.

Citações: simples e completas

Citações simples

Utilize a tag < q > para inserir dentro dela a citação. Assim: Já diz o ditado: cavalo dado não olha os dentes

Citações completas

Quando a citação é um pouco mais extensa, como se fosse a reprodução de um parágrafo. A tag é < blockquote >. Assim:

Daniel Goleman escreveu em seu livro que a inteligência emocional é:

a habilidade de controlar as emoções antes da tomada de decisões, o que envolve autoconhecimento e reflexão constante.

Abreviações

Quando for necessário inserir a forma abreviada de um termo e deixar sua forma literal subvisível ao passa do mouse, use-se a tag < abbr title="aqui o termo literal" >. Como exemplo, será exibida a abreviação HTML, mas se passar o mouse por cima dela verá o que cada letra significa.

Listas: Ordenadas e Não-ordenadas (OL e UL)

Listas ordenadas

São aquelas que iniciam com números ou letras. Primeiro, abre a lista com a tag < Ol > e depois inicia a inclusão dos itens com a tag < li >:

Lista de mercado

  1. ovos
  2. arroz
  3. trigo
  4. macarrão
  5. leite

Listas não-ordenadas

São aquelas que iniciam com marcadores. Existem três opções: circle, disc e square. Primeiro, abre a lista com a tag < UL > e depois inicia a inclusão dos itens com a tag < li >:

Lista de tarefas

  • lavar roupas
  • cozinhar
  • lavar pratos
  • varrer o chão
  • lavar o banheiro

Para alterar o tipo de marcação das listas, dentro das tags de abertura, digite type="escolher o tipo de marcação para lista".

Lista de definições

São usadas para incluir um termo, e logo abaixo, sua definição. O termo é incluído com a tag < dt > e a definição com a tag < dd >. A lista é iniciada com a tag < dl >:

Hml
Significa HiperText Markup Language, ou, Linguagem de Marcação de Texto.
